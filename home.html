Remote Education Resource

A Multi-page website for parents facing the challenges of remote education. This Application allows the User to search for educational resources using Education Data Portal and YouTube Data APIs. The User can select criteria such as YouTube channels: Khan Academy, Smithsonian and more. You will also have the option to search for schools in your area by State and Zip.
